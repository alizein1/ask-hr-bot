
Deployment Instructions:

1. Upload all files to a public GitHub repository:
   - askhr.py
   - salaries.xlsx
   - vacation.xlsx
   - Capital_Partners_Code_of_Conduct.txt

2. Go to https://streamlit.io/cloud and deploy a new app using your GitHub repo.
3. Make sure the main file is: askhr.py
4. You and your employees can access it through the public Streamlit link.

Each employee logs in with:
- Their Ecode from salaries.xlsx
- Their 4-digit PIN from the same file

Enjoy!
